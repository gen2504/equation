Теперь каждый раз после того, как ты внесешь в проект иизменение, пропиши следующие команды в консоль в папке проекта:

git add --all (Добавит все файлы проекта в коммит)
git commit -m "message" (Создаст новый коммит, т.е сохранит текущую версию проекта в безопасном месте на компьютере, так что в случае фатального бага или утери файлов, ты сможешь обратиться к любой закомиченной версии проекта)
git push (Запушит твои изменения на гитхаб, так что даже в случае полного слета операционки, твои файлы останутся на их сервере. При работе с новым проектом надо дополнительно указать "-u origin main", чтобы в последующие разы все коммиты пушились в удаленный юрл origin в ветку main)

Текущую версию я уже запушил, можешь посмотреть на сайте.

И, пожалуйста, называй файлы нормально, а то тебе никто денег не заплатит.
